# Verified Apps

Verified Apps is an app signing certificate hash viewer and verifier.\
It checks apps against a built-in internal database of known-good signing fingerprints.

Verified Apps takes an app's package name and signing certificate hash(es) and compares them to entries in the internal database.\
You can share or copy verification info for use outside the app. If an app is not in the database, you can open a pre-filled [GitHub submission issue](https://github.com/privacyguides/verified-apps/issues/new?template=app-submission.yml) in your browser.
