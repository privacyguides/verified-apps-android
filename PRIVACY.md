# Privacy Policy

This app does not store any data permanently other than preferences. This app does not make network connections itself. When you choose to submit an app that is not in the internal database, the app opens your web browser to a GitHub issue form with verification fields pre-filled from data already shown on screen; submitting that form is handled by GitHub in your browser.

This app can see which apps are installed on your device for the purpose of viewing information about them and checking them against the internal database. You can share and copy verification information to other apps using the share/copy buttons. Clipboard content is only used when you tap copy; it is not read otherwise.

You can provide an apk file to this app which will be temporarily stored in this app's cache directory and deleted as soon as it is done processing the needed information about the apk file.
